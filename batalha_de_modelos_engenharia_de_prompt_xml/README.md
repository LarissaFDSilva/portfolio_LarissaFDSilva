Projeto: Batalha de Modelos & Engenharia de Prompt (XML)
Este repositório contém a documentação e os resultados de um experimento prático focado em Engenharia de Prompt e análise comparativa de grandes modelos de linguagem (LLMs).

📋 Descrição do Projeto
O objetivo desta atividade foi avaliar a capacidade de diferentes ferramentas de Inteligência Artificial em processar e executar instruções complexas através de um Prompt Estruturado em XML. O desafio consistiu na criação de uma página HTML Single Page, com CSS integrado, sob o tema da Força Aérea Brasileira, incluindo um sistema de alistamento militar funcional.

🛠️ Especificações Técnicas do Prompt
A estrutura XML utilizada definiu diretrizes rígidas para garantir a padronização dos testes:
Objetivo: Criação de página única em HTML5 e CSS3.
Diretrizes de Design: Layout responsivo, minimalista, paleta de cores em tons de azul e tipografia mista (Sans-serif e Serif).
Funcionalidades Obrigatórias: Menu de navegação por âncoras, seção de portfólio/galeria e formulário de inscrição para conscritos (incluindo medidas corporais e dados de saúde).

🧠 Reflexão Crítica
Melhor Compreensão: O modelo Claude destacou-se pela melhor interpretação da estrutura XML.
Eficiência vs. Qualidade: Observou-se uma grande variação na verbosidade; enquanto o Grok utilizou um alto número de tokens para um resultado básico, o Claude entregou um resultado superior e satisfatório.
Veredito: O Claude foi selecionado como a ferramenta ideal tanto para prototipagem rápida quanto para o desenvolvimento de códigos complexos com base nesta experiência.
